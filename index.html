Pebolim Info

Game Instructions

  1. Open the game on one browser tab/device and pick Host Match for online play.
  2. For each remote player: select a rod, create an offer, send that JSON to the joiner.
  3. Joiners choose Join Match, paste the offer, create an answer, and send it back to host.
  4. Host pastes each answer and applies it to connect players one by one.
  5. First team to 7 goals wins.

Controls

  • Joiner controls assigned rod with W / S.
  • Host controls one selected rod with W / S.
  • Spin controlled rod with Space.
  • Reset Match restarts score and ball position.
